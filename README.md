# 04-Code-Quiz ![Static Badge](https://img.shields.io/badge/MIT_License-grey)
## Description
This GitHub Repository contains code creating a javascript code quiz. It uses HTML Javascrip and CSS to create a quiz with a timer.
## Table of Contents
- [Installation](#installation)
- [Usage](#usage)
- [Credits](#credits)

## Installation
The repository can be found at: https://github.com/jlipovetz/04-Code-Quiz.
## Usage
The page itself is very easy to use. The user just needs to click on "Start Game" and the quiz and timer will start. When the quiz is over they will be presented with a score.
